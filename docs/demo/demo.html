Создайте базу данных под названием «регистрация» . В базе данных регистрации добавьте таблицу с именем users . Таблица пользователей будет содержать следующие четыре поля.

⦁ идентификатор
⦁ имя пользователя - varchar(100)
⦁ электронная почта — varchar(100)
⦁ пароль - varchar(100)

Вы можете создать это с помощью клиента MySQL, такого как PHPMyAdmin.

Или вы можете создать его в командной строке MySQL, используя следующий сценарий SQL:

CREATE TABLE `users` (
  `id` int(11) NOT NULL AUTO_INCREMENT PRIMARY KEY,
  `username` varchar(100) NOT NULL,
  `email` varchar(100) NOT NULL,
  `password` varchar(100) NOT NULL
) ENGINE=InnoDB DEFAULT CHARSET=latin1;

И это все с базой данных.
